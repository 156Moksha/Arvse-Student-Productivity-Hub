# Arvse - Student Productivity Hub 🌱

Arvse is a student productivity and self-growth website designed to help students organize tasks, track goals, improve focus, and manage their learning journey.

## ✨ Features

- 📊 Dashboard
- 🎯 Goal Tracking
- ✅ To-Do List
- ⏰ Pomodoro Timer
- 🔔 Reminders
- 📈 Progress Tracking
- 📚 Learning Resources
- 💭 Daily Quotes

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 👥 Team Project

Arvse was developed collaboratively by a team to create a simple productivity platform for students.

## 🎯 Purpose

The goal of Arvse is to provide a calm and organized workspace where students can plan, focus, and improve their skills.

## 📂 Project Structure

- Member A - Dashboard and integration
- Member B - Goals and To-Do modules
- Member C - Pomodoro, Reminders, Quotes
- Member D - Progress and Resources

## 🚀 How to Run

1. Download or clone this repository
2. Open the project folder
3. Open HTML files in a browser or use Live Server
